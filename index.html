manifest.webmanifest Note : Le Calendrier du Village est un calendrier vivant. Il est régulièrement enrichi et complété avec de nouvelles informations traditionnelles, culturelles et communautaires.
Installer : Android (Chrome) → menu ⋮ → « Ajouter à l’écran d’accueil ». iPhone (Safari) → Partager → « Sur l’écran d’accueil ».

Les événements proviennent de /data/events.json (généré depuis votre feuille Excel). Le fichier data/calendrier.ics est disponible pour l’import dans Android/iPhone.
